Sprites are loaded with three values delimited by tabs

1. Sprite Name. 2. Y coordinate (double) 3. X coordinate (double)

If coordinates are out of bounds or a name which isn't loaded is written, the sprite sheet won't be loaded.

Sprite Names:
Well = Well
Vines = Vines
Pillar = Pillar
Kni = Knight
Tree = Tree
Barrel = Barrel
ETable = Empty Table
Table = Table
Light = Light
Key = Key
Skel = Skeleton
EnemyU = Enemy facing up/north direction
EnemyL = Enemy facing left/west direction
EnemyR = Enemy facing right/east direction
EnemyD = Enemy facing down/south direction
